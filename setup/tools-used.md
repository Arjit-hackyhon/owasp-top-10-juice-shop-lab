# Tools Used

## OWASP Juice Shop
- Intentionally vulnerable web application
- Used for demonstrating OWASP Top 10 vulnerabilities

## Burp Suite Community Edition
- Used for intercepting HTTP requests
- Used for Intruder brute force testing
- Used for analyzing application traffic

## Chromium Browser
- Used for interacting with the application
- Developer tools used for source code analysis

## Node.js & NPM
- Used to install and run Juice Shop application
