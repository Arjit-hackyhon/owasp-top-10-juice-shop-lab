# Installation Steps

## Step 1: Install Node.js

Download and install Node.js from official website.

Verify installation:


node -v
npm -v


## Step 2: Install OWASP Juice Shop


npm install -g juice-shop


OR


git clone https://github.com/juice-shop/juice-shop.git

cd juice-shop
npm install


## Step 3: Start Application


npm start


Application will start at:

http://localhost:3000

## Step 4: Install Burp Suite

Download Burp Suite Community Edition and configure browser proxy to 127.0.0.1:8080
