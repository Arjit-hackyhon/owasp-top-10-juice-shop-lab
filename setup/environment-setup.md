# Environment Setup

## System Requirements

- Operating System: Windows 10 / Linux / macOS
- RAM: Minimum 8 GB Recommended
- Disk Space: At least 5 GB free
- Internet Connection Required

## Lab Environment

This lab was performed in a controlled local environment using:

- Localhost server (127.0.0.1)
- OWASP Juice Shop application running on port 3000
- Burp Suite Community Edition for traffic interception
- Chromium browser for testing

## Network Configuration

- Burp Proxy configured to intercept HTTP traffic
- Browser proxy set to 127.0.0.1:8080
- Application accessible at http://localhost:3000

⚠ This testing was performed in a legal lab environment for educational purposes only.
