# 🔐 OWASP Top 10 – Juice Shop Practical Lab

## 📌 Project Overview

This repository demonstrates practical exploitation of the OWASP Top 10 vulnerabilities using OWASP Juice Shop.

The project includes:

- Detailed vulnerability reports
- Step-by-step reproduction steps
- Screenshot evidence
- Risk analysis
- Mitigation strategies
- Practical attack demonstrations using Burp Suite

---

## 🛠 Lab Environment

- Application: OWASP Juice Shop
- Proxy Tool: Burp Suite Community Edition
- Runtime: Node.js
- Browser: Chromium
- Localhost Environment (127.0.0.1)
